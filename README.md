
# 🧺 Laundry Mart

A simple and responsive HTML-based landing page for a fictional laundry service. This project was built as part of my MERN Stack learning journey to practice the fundamentals of HTML and webpage structure.

## 📖 About

Laundry Mart is a demo website that showcases a laundry service. It includes an introduction to the business, available services, a pricing table, a booking form, and contact information.

This project helped me understand how to structure web pages using semantic HTML elements.

## ✨ Features

- 🏠 Welcome section
- 📄 About Laundry Mart
- 🖼️ Laundry service illustration
- 🧺 Services list
- 💰 Price list table
- 📝 Booking form
- 📞 Contact information
